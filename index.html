An Analyzable Learning Record Store

Get insight into your students’ progress

Increasingly, students are asked to read web pages, do online tests, watch video’s and look at slideshows. We can check if they did do those things, but we can’t see when, where, how long they took and how many tries it took to complete.

That’s why we built a simple interface to aggregate, manage and analyze these learning records. This happens using the Tin Can API, an open standard for saving learning data. A lot of learning tools already implement this, which makes it easy for you to connect to it.

Start generating reports in minutes

1 Create a Record Store

You need a place to store all that data. Create an account, choose your Record Store name and you are ready to roll! This Record Store can be used for multiple Events, so you won’t ever need to repeat this step.

2 Create an Event Link

An Event can be an online test, visiting a web page or watching a video. Create an Event from the Dashboard, share the link with your students and wait for the data to come rolling in...

3 Build a Report

You can see results, duration and simular information for each student, but it gets really interesting once you start graphing it! Pine comes with some default Reports, so the pretty graphs are only a few clicks away.
